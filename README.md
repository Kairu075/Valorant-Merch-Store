Valorant Merch Online Store

1. Project Overview

This project is a Valorant-themed online merchandise website created as an individual activity for the Platform Technologies course. The website showcases various Valorant-inspired items such as apparel and gaming accessories, with interactive features implemented using JavaScript functions and events.

The project demonstrates basic front-end web development concepts including HTML structure, CSS styling, and JavaScript interactivity, and is deployed online using a free web hosting service.

2. Features

-Responsive navigation menu
-Product listing with ratings
-Product details page with quantity selection
-Interactive buttons using JavaScript events
-Hover effects on partner logos
